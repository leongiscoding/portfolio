# Portfolio — Edwin Tan

A single-page personal portfolio site for **Edwin Tan**, built
on Next.js 16 and React 19. The site brings together a custom WebGL hero
canvas, GSAP-driven section reveals, smooth scrolling, and a bespoke cursor /
preloader system, designed to feel like a small interactive product rather
than a static résumé.

The repository is also a worked example of an **AI-assisted development
workflow**: every step from scaffolding to deployment is driven through
Claude Code / Claude Desktop, with branching, linting, smoke testing, and
merging automated by project-level hooks and slash commands.
